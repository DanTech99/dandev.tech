## Getting Started



### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/dantech99/dantech.dev
   ```
2. Install NPM packages
   ```sh
   npm install

   pnpm install
   ```

3. Running project
    ```
    pnpm run dev
    pnpm astro dev
    ```
